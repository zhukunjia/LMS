### 用户信息
初始化了6个用户，分别是`test1`、`test2`、`test3`、`test4`、`test5`、`admin`，密码和用户名相同。 其中`test1`、`test2`、
`test3` 加入了`USER`权限组，`admin`加入了`MANAGE`权限组，即`admin`具有图书管理模块所有的权限，`test1`、`test2`、`test3`
只有查询权限。

### 接口使用
- 登录接口
  - 请求方式：POST
  - 请求地址：http://localhost:8080/v1/user/login
  - 请求头：
  - 无
  - 请求体：{
    "userName": "admin",
    "password": "admin"
    }

- 新增图书接口
  - 请求方式：POST
  - 请求地址：http://localhost:8080/v1/user/login
  - 请求头：
      - x-lms-userId:U2054744230764613632
      - x-lms-token:caf4bcfaaa044398b46748adf3f6ba60
  - 请求体：{
    "name": "Java手册",
    "isbn": "22",
    "author": "小马",
    "publisher": "人民出版社",
    "publishTime": "2015-06-13",
    "categoryName": "教育",
    "cover": "",
    "price": 12.3
    }

- 编辑图书接口
  - 请求方式：POST
  - 请求地址：http://localhost:8080/v1/library/update
  - 请求头：
      - x-lms-userId:U2054744230764613632
      - x-lms-token:caf4bcfaaa044398b46748adf3f6ba60
  - 请求体：{
    "id": "L2055101761752010752",
    "name": "Java手册更新"
    }

- 删除图书接口
  - 请求方式：DELETE
  - 请求地址：http://localhost:8080/v1/library/{libraryId}
  - 请求头：
      - x-lms-userId:U2054744230764613632
      - x-lms-token:caf4bcfaaa044398b46748adf3f6ba60
  - 请求参数：
      - libraryId：图书id

- 批量删除图书接口
  - 请求方式：DELETE
  - 请求地址：http://localhost:8080/v1/library/batchRemove
  - 请求头：
      - x-lms-userId:U2054744230764613632
      - x-lms-token:caf4bcfaaa044398b46748adf3f6ba60
  - 请求体：[
    "L2054775828251807744",
    "L2054775452186316800"
  ]

- 查询图书详情接口
  - 请求方式：GET
  - 请求地址：http://localhost:8080/v1/library/{libraryId}
  - 请求头：
      - x-lms-userId:U2054744230764613632
      - x-lms-token:caf4bcfaaa044398b46748adf3f6ba60
  - 请求参数：
      - libraryId：图书id

- 批量删除图书接口
  - 请求方式：POST
  - 请求地址：http://localhost:8080/v1/library/pageQuery
  - 请求头：
      - x-lms-userId:U2054744230764613632
      - x-lms-token:caf4bcfaaa044398b46748adf3f6ba60
  - 请求体：{
    "pageSize": 10,
    "name": "Java"
    }
